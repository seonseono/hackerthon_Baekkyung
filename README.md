# 제1회 백경 해커톤 관리자 대시보드 개발</span>

<br/>
운전자 이상행동 탐지 프로젝트 고도화의 방안으로 관리자가 쉽게 확인할 수 있는 Monthly Report 대시보드를 개발했습니다.   
사이드바에서 운전자 ID를 선택하면 해당 운전자의 주행 중 전송된 알람 횟수, 로그 데이터 등을 확인할 수 있습니다.
